# Compiled game discs for the Acornsoft Elite Demonstration Disc

This folder contains the SSD disc image for the Acornsoft Elite Demonstration Disc, as produced by the build process. This SSD image can be loaded into an emulator like JSBeeb or BeebEm, or into a real BBC Micro using a device like a Gotek.
